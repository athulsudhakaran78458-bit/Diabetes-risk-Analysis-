🩺 Diabetes Risk Analysis using Exploratory Data Analysis and Statistical Hypothesis Testing

📌 Project Overview

This project explores the Pima Indians Diabetes Dataset to identify the key health factors associated with diabetes. The analysis combines Exploratory Data Analysis (EDA), statistical hypothesis testing, correlation analysis, and data visualization to understand relationships between clinical variables and diabetes risk. The findings provide valuable insights for future predictive machine learning models.

---

🎯 Objectives

- Analyze the distribution of clinical variables related to diabetes.
- Identify significant factors influencing diabetes risk.
- Perform Exploratory Data Analysis (EDA).
- Conduct statistical hypothesis testing.
- Examine correlations between predictor variables and diabetes outcome.
- Provide insights for future machine learning classification models.

---

📂 Dataset

Dataset: Pima Indians Diabetes Dataset

Source: National Institute of Diabetes and Digestive and Kidney Diseases (NIDDK) via UCI Machine Learning Repository / Kaggle.

Features

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Diabetic / Non-Diabetic)

📊 Key Findings

- Glucose is the strongest predictor of diabetes.
- BMI, Age, and Pregnancies are statistically significant risk factors.
- Blood Pressure was not statistically significant in this dataset.
- The dataset contains biologically implausible zero values that should be treated as missing data.
- Class imbalance should be addressed before developing predictive models.

🚀 Results

- Glucose showed the highest correlation with diabetes outcome.
- Statistical tests confirmed significant differences in Glucose, BMI, Age, and Pregnancies between diabetic and non-diabetic groups.
- Visualizations clearly demonstrated the relationship between glucose levels and diabetes risk.
- The project provides a strong foundation for developing machine learning models such as Logistic Regression, Random Forest, and XGBoost.
